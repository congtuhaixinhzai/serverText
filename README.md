# 📌 Giới Thiệu

**serverText** là hệ thống lưu trữ và đồng bộ dữ liệu server dạng text/json đơn giản, nhanh và dễ tích hợp cho game, launcher hoặc ứng dụng tự động hóa.

Hỗ trợ:
- ⚡ Đồng bộ dữ liệu realtime
- 🌐 Đọc dữ liệu trực tiếp từ GitHub Raw
- 📦 Nhẹ, dễ triển khai
- 🔒 Hoạt động ổn định
- 🛠 Dễ tích hợp cho Java / Android / LibGDX / Python
